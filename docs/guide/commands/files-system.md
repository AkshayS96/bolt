# 📁 Files & System

| Command | Description |
|---------|-------------|
| `bolt file hash <path>` | SHA256 hash of file |
| `bolt file size <path>` | File size |
| `bolt file lines <path>` | Count lines |
| `bolt file stats <path>` | File metadata |
| `bolt diff <file1> <file2>` | Colored file diff |
