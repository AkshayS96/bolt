# 🔑 ID Generators

| Command | Description |
|---------|-------------|
| `bolt uuid` | Generate a UUID v4 |
| `bolt uuid short` | Short unique ID (8 chars) |
| `bolt nanoid` | Generate a NanoID |
| `bolt cuid` | Generate a CUID-like unique ID |
| `bolt random [length]` | Random alphanumeric string |
